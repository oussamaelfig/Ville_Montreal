API Documentation

This is the documentation for the InspectionMTL API.

Search API

The Search API allows you to search for contrevenants based on the following criteria:

  • etablissement - Name of establishment
  • proprietaire - Name of owner
  • rue - Street name

To use the Search API, send a POST request to /search with the following parameters:

  • etablissement (optional) - Name of establishment to search for
  • proprietaire (optional) - Name of owner to search for
  • rue (optional) - Street name to search for

Contrevenants API

The Contrevenants API allows you to retrieve a list of contrevenants who committed an infraction between two dates.

To use the Contrevenants API, send a GET request to /contrevenants with the following parameters:

  • du - Start date (ISO 8601 format)
  • au - End date (ISO 8601 format)
